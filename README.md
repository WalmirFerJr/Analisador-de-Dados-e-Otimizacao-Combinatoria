# Analisador-de-Dados-Espaciais-e-Otimiza-o-Combinat-ria

Projeto desenvolvido na disciplina de Introdução e Análise de Algortimos da USP

• Implementei um software de processamento de dados geoespaciais em C para calcular profundidades e volumes baseados em coordenadas de entrada.

• Desenvolvi um motor de busca recursivo para identificar agrupamentos de dados em grandes matrizes, automatizando o cálculo de métricas específicas para cada região identificada. 

• Criei um algoritmo de tomada de decisão que seleciona os melhores componentes de um conjunto para satisfazer uma restrição de capacidade total, utilizando lógica de programação recursiva.
